# Superstore - Business Analysis Project
The screemshot image is final dashboard of power bi, note this before you download 
Project Goal: "Analyzing Superstore sales to identify seasonal trends and profit drivers, the full project file is attached as word document [Starter_BA_Project_Retail_Sales.docx](https://github.com/user-attachments/files/27711546/Starter_BA_Project_Retail_Sales.docx)
This repository contains files of Power BI dashboards which were analyzed using Excel from the point of view of shareholders, addressing crucial insights and providing remarkable feedback
The .pbix or .pibip file for Power BI is an editable file; learners can download the file along with a Word document for guiding yourself through the project
The xslx or CSV file is an excel file, and a secondary data found on Kaggle (sample superstore); it has been cleaned with the existing co-pilt for temporary use, leaving the rest for users to perform ETL as per their needs
While navigating yourself through Excel, users can find 4 different sheets built in order to address 4 different questions in the report (word); these can be neglected if you start with a table in Power BI
The power dashboard has no relationships built or is directly connected with xslx file, but is imported and updated with hierarchy (check the order date)
